# Mini App — Список записей

## Что это

Telegram Mini App для просмотра, редактирования и удаления записей из записной книжки. Открывается кнопкой «📁 Записи» вместо ссылки на Drive.

## Точка входа

Кнопка «📁 Записи» в reply keyboard меняется с текстовой ссылки на Drive на `web_app`-кнопку, открывающую `https://zapisnaya-knizhka.onrender.com/app`. Ссылка на Drive переходит в ⚙️ Настройки.

## Хостинг

Один HTML-файл `backend/webapp/index.html`, отдаётся Express-сервером по пути `GET /app`. Отдельного деплоя нет — всё на том же Render.

## Аутентификация

Telegram передаёт `window.Telegram.WebApp.initData` при открытии Mini App. Каждый API-запрос включает этот строку в заголовке `X-Telegram-Init-Data`. Сервер проверяет HMAC-SHA256 подпись с `BOT_TOKEN` и извлекает `user.id`. Если подпись невалидна — 401.

## API-эндпоинты

Все требуют заголовок `X-Telegram-Init-Data`.

| Метод | Путь | Описание |
|-------|------|----------|
| `GET` | `/api/records` | Все записи пользователя |
| `PATCH` | `/api/records/:id` | Обновить запись (body: `{data, comment}`) |
| `DELETE` | `/api/records/:id` | Удалить запись |

Данные читаются/пишутся в Drive пользователя через существующий `readJson`/`writeJson`.

## UI

### Шапка
- Слева: «Записи» (заголовок)
- Справа: количество записей

### Фильтры
Горизонтальный скролл с pill-кнопками: «все» + категории которые реально есть в базе. «все» активна по умолчанию.

### Список
Каждая строка в свёрнутом виде:
```
Имя / название          ×
суть · контакт
```

Тап на строку — раскрывается форма редактирования со всеми полями из `data{}` + отдельное поле «комментарий» (курсивом, необязательное). Кнопка «Сохранить» отправляет `PATCH`.

Крестик `×` — удаляет запись (`DELETE`) с анимацией fade-out. Без подтверждения.

### Дизайн
Чёрно-белый. Шрифт системный (-apple-system). Фон белый, разделители светло-серые. Форма на сером фоне (#f2f2f2). Кнопка «Сохранить» — чёрная.

## Состояния

- **Загрузка** — спиннер по центру
- **Пустой список** — текст «Ещё ничего не сохранено»
- **Ошибка загрузки** — текст «Не удалось загрузить записи»
- **Сохранение** — кнопка «Сохранить» показывает «...», после успеха строка сворачивается
